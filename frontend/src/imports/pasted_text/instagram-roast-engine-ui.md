Instagram Roast Engine – Figma UI Design Prompt

Design a responsive web application for an Instagram Profile Roast Engine using Neo Brutalism combined with soft pastel pink aesthetics, internet-core graphics, and meme culture.

Project Concept

The user enters any public Instagram username.

The website scrapes publicly available profile information (bio, profile picture, followers, following, posts, captions, etc.) and generates a hilarious roast.

The experience should feel like someone made a professional cyberbullying machine—but in the cutest way possible.

The website should be sarcastic, playful, chaotic, handcrafted, and intentionally over-the-top.

Avoid making it look like a normal SaaS dashboard.
